## 今後の改良案

### 1. Hero を再設計する

- 体験価値を一言で言い切るコピーへ寄せる
- `初心者歓迎` `予約制` `所要時間` など即断材料を Hero 近くへ置く
- 手元、素材、空間の魅力が出る構成に強化する

### 2. 体験価値をもっと具体化する

- 何種類の香りに触れられるのか
- どこまで自分で選べるのか
- どんな形で持ち帰れるのか
- ギフト、記念、自己表現など利用シーンを具体化する

### 3. Program 比較を強くする

- `時間 / 価格 / 向いている人 / 持ち帰り内容` を整理する
- 最もおすすめのプランを立てる
- 一覧比較で迷いを減らす

### 4. FAQ / 安心材料を追加する

- 初心者でも大丈夫か
- 何を持っていけばよいか
- 何人で参加できるか
- ギフト相談は可能か

### 5. Smart Scent Design との接続を強める

- 先にデジタル調香を試す意味をもっと明確にする
- 店頭との連動メリットを前面に出す
- オンライン体験から workshop 予約へ流れる理由を作る

### 6. CTA 設計を強める

- LP の途中にも CTA を置く
- 最終 CTA 周辺に安心材料と期待材料を集約する
- 必要なら `人気プラン` や `季節限定` のような即時性演出も検討する

## 最高のワークショップLPにするための方向性

- 説明ページではなく、参加したくなる感情を作る
- 香りを選ぶ高揚感と、予約判断に必要な合理情報を両立する
- 読後の理想状態は「理解した」ではなく「参加したい」になること
